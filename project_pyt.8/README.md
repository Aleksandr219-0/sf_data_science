# Угадай число(1–100)

Проект по угадыванию числа, загаданного компьютером, из диапазона от 1 до 100.  
Алгоритм использует двоичный поиск и в среднем находит число менее чем за 10 попыток.

## Как запустить

1. Клонировать репозиторий:
   `git clone https://github.com/username/guess-number.git`
2. Установить зависимости:
   `pip install -r requirements.txt`
3. Запустить:
   `python game_`

## Алгоритм

Число угадывается методом бинарного поиска: на каждом шаге диапазон возможных значений делится пополам.
